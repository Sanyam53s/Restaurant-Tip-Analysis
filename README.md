# 🍽️ Restaurant Tip Analysis

## 📌 Project Overview
This project analyzes restaurant tipping behavior using Python. The goal is to identify patterns in tips based on factors like total bill, gender, day, and time.

---

## 📊 Dataset
The dataset contains information about:
- Total bill amount
- Tip given
- Gender
- Smoking status
- Day of the week
- Time (Lunch/Dinner)
- Size of the group

---

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Key Analysis Performed
- Tip vs Total Bill relationship
- Gender-wise tipping behavior
- Day-wise and time-wise analysis
- Smoking vs Non-smoking comparison
- Group size impact on tips

---

## 🔍 Key Insights
- Higher total bill generally leads to higher tips
- Dinner time has higher tipping compared to lunch
- Weekend tips are higher than weekdays
- Group size affects tipping patterns
